# Roadmap

## Phase 1: Prototype

- Build SwiftUI shell.
- Use sample thread fixtures.
- Implement static triage categories.
- Show summary, confidence, and suggested action.

## Phase 2: Import Workflow

- Add CSV/TXT import.
- Normalize exported message threads.
- Detect large threads by message count and recent unread activity.
- Add search, filters, and category review.

## Phase 3: Local Intelligence

- Add rule-based first-pass classification.
- Add local summarization adapter.
- Extract dates, actions, links, addresses, and commitments.
- Store summaries in a local encrypted index.

## Phase 4: macOS Messages Advanced Mode

- Research direct local Messages indexing.
- Add permission onboarding.
- Detect unread or recently active threads.
- Handle schema changes and indexing failures gracefully.

## Phase 5: Product Hardening

- Add privacy dashboard.
- Add cloud AI opt-in mode.
- Add exportable reports.
- Add feedback-based tuning.
- Prepare notarized distribution or App Store review strategy.
